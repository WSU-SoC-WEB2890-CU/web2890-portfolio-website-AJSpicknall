The Hobby Tracker

A course project published to Netlify with source code available on GitHub.

Project Snapshot

Course/Context: WEB 3350

Role: UX designer, interface developer

Tools: Figma, HTML, SCSS, JavaScript

Live Demo: web3350-exercise-2.netlify.app

GitHub: github.com/AJSpicknall/WEB3350-Exercise2

Challenge

We wanted to make something where people could publish their favorite events and share them with others in one easy place.

Solution

We built an input box flow that lets a user fill in event details and post them so each new item appears below.

Result

The final result worked efficiently and made sharing event content fast and straightforward.
