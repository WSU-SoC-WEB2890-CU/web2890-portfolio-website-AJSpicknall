Buy and Car Site

Project site centered on browsing, comparing, and exploring car listings in a clean layout.

Project Snapshot

Course/Context: WEB 2890

Role: Interaction designer and developer

Tools: HTML, SCSS, JavaScript

Live Demo: web2890-ajs-project-1.netlify.app

GitHub: github.com/WSU-SoC-WEB2890-CU/web2890-project-1-AJSpicknall.git

Challenge

Customers faced too many competing product details and unclear call-to-action hierarchy.

Solution

Introduced structured information zones, progressive disclosure for specifications, and stronger action buttons.

Result

Prototype analytics showed more completed purchase paths and fewer bounces from product pages.
