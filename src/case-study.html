Case Study:
Old Videogames Nostalgia Hub

A UX-focused redesign of a retro gaming forum where users could rediscover classics, share memories, and find helpful threads faster.

Project Overview

Role: UX designer and front-end developer

Timeline: 8 weeks

Goal: Improve discussion discovery and thread participation for users looking for old video game communities and nostalgia content.

Problem Statement

Users reported that nostalgic game discussions were buried across old threads, off-topic posts, and inconsistent categories. Most visitors struggled to find active conversations about specific systems, game eras, and genres without excessive scrolling.

Research and Insights

  • Interviewed 10 retro gaming forum users across different age groups.
  • Ran a competitive analysis of 4 gaming forums and community archive sites.
  • Found that better thread filtering, tagging, and saved-topic tracking were top priorities.

Design Process

  1. Mapped user flows for browsing, filtering, and multiple thread tasks.
  2. Created low-fidelity wireframes focused on category clarity and thread information gathering.
  3. Built an interactive prototype and tested it with 6 users.
  4. Updated navigation labels, tag structure, and search behavior based on test feedback.

Accessibility Decisions

  • Improved contrast beyond WCAG AA target for body and action text.
  • Added strong focus indicators and logical keyboard usage.
  • Used proper tools for easier screen reader navigation.

Outcome

Usability testing after inputting my iterations showed a 39% reduction in time to find a relevant thread and post a reply in the correct section. Participants described the final version as easier to scan and less overwhelming on mobile.

Reflection

This project reinforced the value of pairing nostalgic visual style with clear community workflows. The strongest improvements came from testing early and refining the forum before full development.
